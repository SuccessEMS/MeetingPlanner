Access and View MeetingPlanner Web Scheduler

Outlook/Exchange and other MeetingPlanner applications.

  1. Open the Web Scheduler with the URL provided by the Systems Administrator (ie. http://yourserver/suite).
  2. Select the calendar icon to open the Web Scheduler application.
  1. Select from three calendar views: 1 (day), 5 (week), or 28 (month).

    

Note: Reservations can be edited but cannot be created from the monthly view.
