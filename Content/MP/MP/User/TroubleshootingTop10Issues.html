Top Meeting Planner Issues (and How to Fix Them)

  1. Changing room booked in Web Scheduler removes Outlook appointment instead of updating to new room location
    1.  Users should be booking the new room first, and only then deleting the original room.
  2. Customer reports blank white screen when booking in the Web Scheduler
    1. Get the Web.log file from the app server, in C:/Temp/MeetingPlannerLogs
  1. Email notifications are not sending. (If they report that the keywords are not rendering correctly, despite having confirmed that the keyword is accurate according to the available keywords for the template in question, then… we know. Times are also off for some of them.)
  2. What systems do you work on?
    1. YES
      1. 32-bit Outlook 2010
      2. 32-bit Outlook 2013
      3. Exchange 2013
      4. Exchange 2010
      5. Office 365 (Hosted Exchange)
      6. Chrome
    2. NO
      1. 64-bit Outlook (any year)
      2. Exchange 2007
      3. Outlook Web App (incl. accessing Outlook 365 email via the Web Interface)
      4. Internet Explorer (Web Scheduler doesn’t work at all in IE8)
      5. Firefox
