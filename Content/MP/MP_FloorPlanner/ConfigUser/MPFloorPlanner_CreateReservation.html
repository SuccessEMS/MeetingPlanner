Create Reservation

Using Forms Authentication

  1. Enter your Windows log in credentials.

  1. Use calendar to change date of reservation.
  2. Use arrows to edit time of reservation.

  1. Select Reserve.

Using MeetingPlannerTouch

  1. Select an Available room. Once a room is selected, the user is automatically taken to the MeetingPlannerTouch (MPTouch) screen.
  2. Select No Reservations from the Vacant Schedule window.

  1. Select Available from Select a Free Time Slot window. FloorPlanner defaults and displays room availability for current day.

  1. Select a description of the type of meeting from Select a Title window.

  1. Enter Username from Confirm Reservation window. MP credentials (e.g., Ted.Smith).
  2. Enter Password from Confirm Reservation window.

MP Credentials

  1. Select Return or Save.
  2. Verify the following fields from the Confirm Reservation window:
  • Description—edit by selecting Descriptionfield. Enter desired description.
  • Start—edit by selecting Start field. Use +/- to adjust start time. Set start time. If user accepts default start time, FloorPlanner automatically reserves meeting room within the next 5 minute increment from the current time.
  • End—edit by selecting End field.  Use +/- to adjust end time. Set end time.
  • Duration—edit by selecting Duration field.
  1. Select Save. The room is reserved in MP unless there is no activity or Save is not selected within the allotted time. If MPTouch “times-out” the reservation is not created.
  2. Select X to exit out of the reservation.

NOTE: All meetings and updates created in FloorPlanner synchronize with MeetingPlanner software.
