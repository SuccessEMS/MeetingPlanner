Exchange Sync, Appointment Fields Accessed in Push

Add

  • Start and End Date
  • Subject—From Meeting Planner Reservation. Creates default one if left blank
  • Body—Will write "Meeting Planner Reservation ID nnn" label only
  • SentOnBehalfOf (often displayed as Organizer)—Meeting Planner sends in the User Name of the Reservation Owner. If not valid, field is left to default value, the mailbox account itself.
  • User Properties, Res_ID and Room_Res_ID

Update

  • Start and or End Date
  • User Properties, Res_ID and Room_Res_ID—Only if not yet in the Appointment
